# Лабораторная работа: Django

## Описание

Учебное web-приложение на Django.

## Используемые технологии

- Python
- Django
- PyMySQL
- HTML
- CSS

## Запуск

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py runserver
```

Адрес приложения:

```text
http://127.0.0.1:8000/
```
