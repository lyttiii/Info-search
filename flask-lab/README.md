# Лабораторная работа: Flask

## Описание

Учебное web-приложение на Flask.

## Используемые технологии

- Python
- Flask
- Flask-SQLAlchemy
- Flask-Login
- Flask-WTF
- WTForms
- PyMySQL
- HTML
- CSS

## Запуск

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Адрес приложения:

```text
http://127.0.0.1:5000/
```
